# 祈绪校园

面向校园场景的综合性服务平台，以微信小程序为移动端入口，提供论坛社区、校园跑腿任务、资源分享、积分运营与钱包结算等能力，并配套 Web 管理后台进行内容与业务运营。

## 项目结构

| 目录 | 说明 |
|------|------|
| `miniprogram-1/` | 微信小程序（用户端） |
| `server-spring/` | 后端 API 服务（Spring Boot） |
| `enw_server/` | 管理后台前端（Vue 3），构建后部署至 `/enw` |
| `archive/legacy-backend/` | 已退役的 Express 旧后端与相关说明 |

## 功能概览

### 微信小程序（用户端）

小程序底部 Tab 分为四个主模块：**论坛**、**任务大厅**、**消息**、**我的**。

#### 校园论坛

- 帖子发布、编辑、浏览与详情展示
- 评论、点赞、收藏与互动消息
- 分类浏览（日常分享、吃瓜吐槽、失物招领等）
- 热帖榜单与搜索
- 个人内容管理（我的发布、点赞、评论、收藏）

#### 任务大厅（校园跑腿）

- 发布与浏览校园任务（取快递、拿外卖、帮忙搬东西等）
- 按服务类型、状态、地点、性别等条件筛选
- 接单员认证与接单流程
- 任务详情、订单跟踪与状态流转
- 我发布的任务 / 我接的任务 / 我的订单管理
- 任务相关钱包收益与提现

#### 消息中心

- 系统通知与互动消息
- 私信聊天（WebSocket 实时通信）
- 未读消息统计与提醒

#### 资源广场

- 浏览与分享校园资源（学习资料、游戏资源、软件资源等）
- 资源详情、审核上架与积分购买
- 我卖出的资源订单管理

#### 积分与签到

- 每日签到与连续签到奖励
- 积分明细与积分商城兑换
- 分享等运营活动积分

#### 个人中心

- 微信授权登录与手机号绑定
- 个人资料、头像与主页背景
- 收货地址管理
- 钱包余额、流水与提现
- 意见反馈与关于页面

### 管理后台（Web）
基于 <a href="https://github.com/Daymychen/art-design-pro">Art Design Pro</a> 模版构建

| 模块 | 主要功能 |
|------|----------|
| **总览** | 工作台、数据统计、审计日志 |
| **消息与用户** | 系统消息推送、用户管理 |
| **社区与内容** | 帖子管理、反馈处理、内容治理、小程序应用管理 |
| **任务与配送** | 接单员认证/管理、任务审核、任务机制、包裹编辑、营销活动、地点管理 |
| **交易与清结算** | 订单管理、提现审核、支付设置 |
| **资源与积分** | 资源审核/管理、积分商城、购买/兑换明细、积分运营 |
| **品牌与触达** | 触达中心、运营位、客服二维码、联系设置 |

## 技术栈

### 微信小程序 `miniprogram-1/`

| 类别 | 技术 |
|------|------|
| 框架 | 微信小程序原生开发（WXML / WXSS / JavaScript） |
| UI | 自定义导航栏、Remix Icon 图标 |
| 通信 | REST API + WebSocket |
| 能力 | 微信登录、手机号授权、分享、内容安全检测 |

### 后端 `server-spring/`

| 类别 | 技术 |
|------|------|
| 语言 / 运行时 | Java 17 |
| 框架 | Spring Boot 3.4 |
| 持久层 | MyBatis-Plus + MySQL |
| 数据库迁移 | Flyway |
| 缓存 | Redis + Redisson（分布式锁） |
| 消息队列 | RabbitMQ（异步入账、Outbox 等） |
| 实时通信 | Spring WebSocket |
| 认证 | JWT（管理端）、微信 OpenID（小程序端） |
| 文件存储 | 阿里云 OSS + Thumbnailator 缩略图 |
| 其他 | Spring Validation、Actuator、AOP 审计日志 |

### 管理后台 `enw_server/`

| 类别 | 技术 |
|------|------|
| 框架 | Vue 3 + TypeScript |
| 构建 | Vite 7 |
| UI 组件 | Element Plus + Tailwind CSS 4 |
| 状态管理 | Pinia（含持久化） |
| 路由 | Vue Router 4 |
| HTTP | Axios |
| 图表 | ECharts |
| 规范 | ESLint、Prettier、Stylelint、Husky |

## 快速启动

### 环境要求

- **后端**：JDK 17+、Maven 3.8+、MySQL 8、Redis、RabbitMQ
- **管理端**：Node.js ≥ 20.19、pnpm ≥ 8.8
- **小程序**：微信开发者工具

默认端口：`3000`（HTTP API）、`30001`（WebSocket）

### 微信小程序

1. 使用微信开发者工具打开 `miniprogram-1/` 目录
2. 在 `utils/config.js` 中配置后端 API 地址
3. 编译预览或上传体验版
